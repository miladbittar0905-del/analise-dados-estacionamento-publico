# Análise de Dados de Estacionamento

## Contexto
Projeto acadêmico de Ciência de Dados com foco na análise operacional e financeira de um estacionamento 24h que atende clientes avulsos e conveniados.

## Objetivos
- Analisar padrões de permanência dos clientes
- Avaliar o impacto dos convênios no faturamento
- Identificar horários de maior e menor ocupação
- Apoiar decisões de precificação com base em dados

## Ferramentas Utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn

## Análises Realizadas
- Análise exploratória de dados (EDA)
- Cálculo de ticket médio e receita total
- Comparação entre clientes avulsos e conveniados
- Análise temporal por dia da semana e horário

## Principais Insights
- Convênios representam a maior parte do fluxo, mas menor participação na receita
- Horários de pico concentrados no final da tarde e início da noite
- Oportunidades de precificação diferenciada em horários de baixa ocupação


## Observação sobre os dados
Os dados utilizados neste projeto são sensíveis e pertencem a uma operação real.
Por esse motivo, o dataset original não está disponibilizado neste repositório.

O notebook apresenta o processo completo de tratamento, análise e geração de insights,
mantendo a estrutura e lógica aplicada aos dados reais.
